# 電磁波と物質 - B2032

福井大学 共通教育科目「電磁波と物質」(B2032) のWeb講義教材です。

## URL

**GitHub Pages:** https://lutelute.github.io/lec_B2032_claude/

## 概要

本教材は、電磁波に関する基礎から応用までを網羅した全15回の講義資料を提供します。技術解説書形式で、数学的導出、物理的解説、コラムを含む詳細な内容となっています。

## 講義内容

### Part 1: 電磁波の基礎（第1回〜第4回）
- 第1回: 電磁波とは
- 第2回: 可視光の色
- 第3回: 電磁波発生と応用
- 第4回: 電磁波と物質との相互作用

### Part 2: スピンと磁気共鳴（第5回〜第8回）
- 第5回: 固体と電子スピン
- 第6回: 磁気共鳴
- 第7回: 電子スピン共鳴
- 第8回: 電子スピン共鳴（応用）

### Part 3: 電磁波の数理（第9回〜第10回）
- 第9回: マクスウェル方程式と波動方程式
- 第10回: 電磁波の伝播・回折

### Part 4: 現代物理への展開（第11回〜第12回）
- 第11回: 量子力学の始まり
- 第12回: 電磁波と特殊相対性理論

### Part 5: 応用と先端研究（第13回〜第15回）
- 第13回: 荷電粒子・プラズマ
- 第14回: ビーミング推進
- 第15回: 光と電波の境目・ミリ波とジャイロトロン

## 機能

- **ダッシュボード**: 学習進捗の管理、講義一覧
- **講義資料**: 詳細な解説、数学的導出、コラム（MathJax対応）
- **インタラクティブ学習**: 波動シミュレータ、スペクトル探索、理解度チェック

## 技術構成

- HTML5 / CSS3
- JavaScript (Vanilla)
- MathJax (数式レンダリング)
- Chart.js (グラフ描画)
- LocalStorage (進捗保存)
