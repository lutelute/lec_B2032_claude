可視光の色

学習目標: 光と色の物理的関係を理解し、物体の色が見える仕組みを説明できるようになる。人間の色覚の仕組み、加法混色と減法混色の違い、散乱現象と色の関係を学ぶ。

2.1 色の物理学

2.1.1 可視光とは

人間の目は電磁スペクトルの極めて狭い領域のみを感知できる。この領域を可視光線（visible light）と呼び、波長約380 nm〜780 nmの範囲に相当する。

なぜこの波長範囲が「見える」のか

太陽の放射スペクトルは約5800 Kの黒体放射に近く、そのピークは可視光領域にある。地球上の生物は、最も豊富に存在する電磁波を利用するよう進化してきた。また、大気の窓（可視光が大気をよく透過する領域）とも一致している。

2.1.2 波長と色の対応

可視光の各波長は特定の色として知覚される：

色 波長範囲 (nm) 周波数範囲 (THz) 光子エネルギー (eV) 代表的波長
紫 380 - 450 670 - 790 2.76 - 3.26 405 nm
青 450 - 495 605 - 670 2.50 - 2.76 470 nm
シアン 495 - 520 575 - 605 2.38 - 2.50 500 nm

例題2.1: 各色の光子エネルギー

問題: 赤色光（波長650 nm）と紫色光（波長405 nm）の光子エネルギーの比を求めよ。

解答:

光子エネルギーは \(E = hc/\lambda\) より、波長に反比例する。

\[ \frac{E_{violet}}{E_{red}} = \frac{\lambda_{red}}{\lambda_{violet}} = \frac{650}{405} = 1.60 \]

紫色光の光子エネルギーは赤色光の約1.6倍である。これが紫外線の方が生体に有害な理由の一つである。

2.1.3 スペクトル色と非スペクトル色

スペクトル色：単一波長の光で表現できる色（虹の色）

非スペクトル色：複数の波長の組み合わせでのみ実現できる色

  • 白：すべての可視光波長を含む（または適切な補色の組み合わせ）
  • マゼンタ：赤と青の組み合わせ。スペクトル上に対応する単一波長がない
  • ピンク：赤と白の混合
  • 茶色：低輝度の橙色

2.2 人間の色覚の仕組み

2.2.1 網膜の構造

人間の網膜には2種類の光受容細胞がある：

細胞 数（片目あたり） 機能 感度 分布
桿体細胞（Rod） 約1.2億個 明暗の検出 極めて高感度（数光子で応答） 周辺視野に多い
錐体細胞（Cone） 約600万個 色の検出 比較的低感度（明所で機能） 中心窩に集中

2.2.2 三色説と錐体細胞

人間の色覚は3種類の錐体細胞（三原色受容体）に基づく：

三種類の錐体細胞
  • S錐体（Short）：ピーク感度 約420 nm（青）
  • M錐体（Medium）：ピーク感度 約530 nm（緑）
  • L錐体（Long）：ピーク感度 約560 nm（赤-黄）

各錐体細胞の吸収スペクトルは広く重なり合っており、単一波長の光でも通常は複数の錐体を刺激する。脳は3種類の錐体からの信号の比を解釈して「色」を知覚する。

導出: 錐体の応答と色知覚

1 錐体の応答関数

各錐体の応答 \(R_i\) は入射光のスペクトル \(I(\lambda)\) と錐体の感度関数 \(S_i(\lambda)\) の積分で表される：

\[ R_i = \int_0^\infty I(\lambda) S_i(\lambda) d\lambda \quad (i = L, M, S) \]
2 メタメリズム（等色現象）

異なるスペクトル分布を持つ2つの光でも、3つの錐体応答 \((R_L, R_M, R_S)\) が同じなら、同じ色として知覚される。これをメタメリズムという。

3 色空間の次元

人間の色知覚は3次元空間で表現できる（3つの独立な錐体応答）。これが「三原色」で全ての色を表現できる理由である。

コラム: 色覚の個人差と色覚異常

色覚には個人差があり、特にL錐体とM錐体の遺伝子はX染色体上にあるため、男性に色覚異常が多い（約8%、女性は約0.5%）。

主な色覚異常の種類：

  • 1型色覚（Protanopia）：L錐体の機能低下。赤系の識別困難
  • 2型色覚（Deuteranopia）：M錐体の機能低下。緑系の識別困難
  • 3型色覚（Tritanopia）：S錐体の機能低下。青系の識別困難（稀）

興味深いことに、一部の女性は4種類の錐体を持ち（テトラクロマシー）、通常より多くの色を識別できる可能性がある。

2.2.3 色の恒常性

人間の視覚系は、照明条件が変化しても物体の色を一定に知覚する能力（色の恒常性）を持つ。例えば、白い紙は太陽光下でも蛍光灯下でも「白」に見える。

これは脳が：

  • 周囲の物体との相対的な色の関係を考慮
  • 照明光のスペクトル特性を推定して補正

を行っているためである。この処理は「ホワイトバランス」と呼ばれ、カメラにも実装されている。

2.3 物体の色が見える仕組み

2.3.1 光と物質の相互作用

光が物体に当たると、以下の現象が起こる：

  • 反射（Reflection）：光が物体表面で跳ね返る
  • 吸収（Absorption）：光のエネルギーが物質に移る
  • 透過（Transmission）：光が物質を通過する
  • 散乱（Scattering）：光が様々な方向に分散する
エネルギー保存則
\[ I_0 = I_r + I_a + I_t \]

入射光強度 = 反射光 + 吸収 + 透過光

比率で表すと: \(R + A + T = 1\)（反射率 + 吸収率 + 透過率 = 1）

2.3.2 物体の色を決める要因

不透明な物体の色は、主に吸収と反射によって決まる。物体は特定の波長を吸収し、残りを反射する。反射された光が目に入り、色として知覚される。

例題2.2: 物体の色と吸収スペクトル

問題: 赤いリンゴが赤く見えるのはなぜか。白色光を照射した場合の反射スペクトルを定性的に説明せよ。

解答:

赤いリンゴの表皮には、青と緑の波長（約400-600 nm）を強く吸収する色素（アントシアニン等）が含まれている。

  • 青色光（450 nm付近）：主に吸収
  • 緑色光（530 nm付近）：主に吸収
  • 赤色光（650 nm付近）：反射

その結果、赤色成分のみが反射されて目に届き、リンゴは「赤く」見える。

同じリンゴを青色光のみで照射すると、青色は吸収されてしまうため、リンゴは黒っぽく見える。

2.3.3 色素と分子構造

物質が特定の波長を吸収する能力は、その分子構造、特に電子の配置に依存する。

導出: 共役系と光吸収

1 電子遷移のエネルギー

分子が光を吸収するとき、電子は基底状態から励起状態へ遷移する。吸収される光のエネルギーは：

\[ E_{photon} = h\nu = E_{excited} - E_{ground} \]
2 共役系と吸収波長

共役二重結合（交互に単結合と二重結合が並ぶ構造）が長いほど、電子の非局在化が進み、HOMO-LUMOギャップが小さくなる。

3 井戸型ポテンシャルモデル

長さ \(L\) の1次元井戸中の電子のエネルギー準位間隔：

\[ \Delta E \propto \frac{1}{L^2} \]

共役系が長い（L が大きい）ほど、\(\Delta E\) は小さくなり、より長波長（低エネルギー）の光を吸収する。

4 応用

β-カロテン（ニンジンの橙色色素）は11個の共役二重結合を持ち、青〜緑色光を吸収して橙色に見える。

2.3.4 構造色

色素に依らない色の発現機構として構造色がある。これは微細構造による光の干渉・回折によって生じる色である。

コラム: 自然界の構造色

多くの生物が構造色を利用している：

  • モルフォ蝶：翅の微細な溝構造による干渉で鮮やかな青色を発現。角度によって色が変化する。
  • 玉虫：多層膜干渉による虹色の輝き
  • 孔雀の羽：2次元フォトニック結晶構造
  • CD/DVD：人工的な構造色（回折格子）

構造色の特徴：

  • 見る角度によって色が変化（イリデッセンス）
  • 色素と違い退色しない
  • 偏光依存性を持つことがある

2.4 光の混色

2.4.1 加法混色（光の三原色）

複数の光を重ね合わせると、それぞれの色が加算されて新しい色が生じる。これを加法混色という。

光の三原色（加法混色）

赤 (Red) + 緑 (Green) + 青 (Blue) = 白

  • R + G = 黄 (Yellow)
  • G + B = シアン (Cyan)
  • B + R = マゼンタ (Magenta)

加法混色の例：

  • テレビ・ディスプレイ（RGB液晶、有機EL）
  • 舞台照明
  • プロジェクター

2.4.2 減法混色（色料の三原色）

色素や絵の具を混ぜると、それぞれが吸収する波長が増え、反射光が減少する。これを減法混色という。

色料の三原色（減法混色）

シアン (C) + マゼンタ (M) + 黄 (Y) = 黒（理論上）

  • C + M = 青
  • M + Y = 赤
  • Y + C = 緑

減法混色の例：

  • カラー印刷（CMYK：CMY + 黒(K)）
  • 絵の具の混合
  • カラーフィルター

導出: 加法混色と減法混色の関係

1 補色の関係

加法混色の原色と減法混色の原色は補色関係にある：

  • 赤 ↔ シアン（赤を吸収）
  • 緑 ↔ マゼンタ（緑を吸収）
  • 青 ↔ 黄（青を吸収）
2 フィルターの効果

シアンフィルターは赤を吸収し、緑と青を透過する。白色光をシアンフィルターに通すと：

\[ I_{out}(\lambda) = I_{white}(\lambda) \times T_{cyan}(\lambda) \]

ここで \(T_{cyan}\) は赤色領域で0に近く、青・緑領域で1に近い。

3 複数フィルターの重ね合わせ

シアンとマゼンタのフィルターを重ねると：

\[ I_{out} = I_{white} \times T_{cyan} \times T_{magenta} \]

シアンは赤を、マゼンタは緑を吸収するため、青のみが透過する。

例題2.3: プリンターのインク混合

問題: カラープリンターでオレンジ色を印刷するとき、CMYKのどのインクをどのような比率で使用するか説明せよ。

解答:

オレンジ色 ≈ 赤 + 黄（加法混色的に考えると）

減法混色では：

  • 黄(Y)インク：青を吸収、赤・緑を反射
  • マゼンタ(M)インク：緑を吸収、赤・青を反射

Y + 少量のM を混合すると：

  • Yが青を吸収
  • Mが緑の一部を吸収
  • 結果：赤が強く、緑が弱い → オレンジ

実際の印刷では Y: 50-100%、M: 30-60%、C: 0%、K: 0% 程度の組み合わせが使われる。

2.5 光の散乱と色

2.5.1 レイリー散乱

光の波長より十分小さい粒子（気体分子など）による散乱をレイリー散乱という。散乱強度は波長の4乗に反比例する。

レイリー散乱の強度
\[ I \propto \frac{1}{\lambda^4} \]

短波長（青）の光は長波長（赤）の光より強く散乱される

導出: レイリー散乱の波長依存性

1 誘起双極子モーメント

入射電磁波の電場 \(\vec{E}\) により、分子に誘起双極子モーメント \(\vec{p}\) が生じる：

\[ \vec{p} = \alpha \vec{E} \]

ここで \(\alpha\) は分極率である。

2 振動双極子からの放射

振動する双極子は電磁波を放射する。放射パワーはラーモアの公式より：

\[ P \propto |\ddot{p}|^2 \propto \omega^4 |\vec{p}|^2 \]
3 波長依存性

\(\omega = 2\pi c/\lambda\) より：

\[ P \propto \omega^4 \propto \frac{1}{\lambda^4} \]
4 数値例

青色光(λ=450nm)と赤色光(λ=700nm)の散乱強度比：

\[ \frac{I_{blue}}{I_{red}} = \left(\frac{700}{450}\right)^4 = 5.8 \]

青色光は赤色光の約5.8倍強く散乱される。

2.5.2 空が青い理由

太陽光が大気中を通過する際、窒素・酸素分子によるレイリー散乱が起こる。青色光は赤色光より強く散乱されるため、空のあらゆる方向から青色光が目に届く。

空が青く見える条件

  • 散乱粒子（分子）のサイズが光の波長より十分小さい
  • 散乱回数が適度である（1回散乱が主）
  • 観測者が散乱光を見ている（太陽を直接見ていない）

2.5.3 夕焼けが赤い理由

太陽が地平線近くにあるとき、太陽光は大気中を長距離通過する。この過程で青色光は散乱により大部分が取り除かれ、赤色光が残って目に届く。

例題2.4: 大気散乱による減衰

問題: 太陽が天頂にあるとき、大気を通過する距離を1とする。太陽高度が10°のとき、大気通過距離は何倍になるか。また、青色光と赤色光の強度比はどう変化するか。

解答:

大気通過距離（エアマス）は近似的に：

\[ AM = \frac{1}{\sin(h)} \]

h = 10°のとき：

\[ AM = \frac{1}{\sin(10°)} = \frac{1}{0.174} = 5.76 \]

大気通過距離は約5.8倍になる。

散乱による減衰は \(I = I_0 e^{-\tau}\) で表され、光学的厚さ \(\tau\) は距離に比例する。

もし天頂での青/赤の強度比が5.8（レイリー散乱の効果で青が多く散乱）だとすると、5.8倍の距離を通過後は：

\[ \frac{I_{blue}}{I_{red}} \propto e^{-5.8 \times 5.8 \times \tau_{blue}/\tau_{red}} \]

青色光は大幅に減衰し、相対的に赤色光が優勢になる。

2.5.4 ミー散乱

粒子サイズが光の波長と同程度の場合（水滴、塵埃など）、ミー散乱が起こる。ミー散乱は波長依存性が弱く、全ての波長がほぼ同様に散乱される。

  • 雲が白い理由：水滴（直径〜10μm）によるミー散乱は全波長を同様に散乱するため、白く見える
  • 霧：同様の理由で白〜灰色に見える
  • 牛乳の白さ：脂肪球（〜1μm）によるミー散乱

2.5.5 チンダル現象

コロイド溶液に光を当てると、散乱により光の筋が見える現象。粒子サイズがレイリー散乱とミー散乱の中間領域にある場合に顕著。

コラム: 月食時の赤い月

皆既月食のとき、月が赤銅色に見える「ブラッドムーン」現象は、地球大気のレイリー散乱で説明できる。

地球の影に入った月には、地球大気を通過して屈折した太陽光のみが届く。この光は長距離の大気を通過するため、夕焼けと同様に青色成分が散乱で失われ、赤色成分が月面を照らす。

月食時の月の色は、地球大気の状態（火山噴火による粒子、雲の量など）によって変化する。

2.6 光の屈折と分散

2.6.1 屈折率の波長依存性

物質の屈折率は光の波長によって異なる（分散）。通常の透明物質では、短波長ほど屈折率が大きい（正常分散）。

コーシーの分散式
\[ n(\lambda) = A + \frac{B}{\lambda^2} + \frac{C}{\lambda^4} + \cdots \]

A, B, C は物質固有の定数。通常 B > 0 なので、短波長ほど n が大きい。

物質 n (青, 486nm) n (黄, 589nm) n (赤, 656nm)
クラウンガラス 1.523 1.517 1.514
フリントガラス 1.664 1.650 1.644
水 1.337 1.333 1.331
ダイヤモンド 2.435 2.417 2.407

2.6.2 プリズムによる分光

プリズムを通過する白色光は、波長ごとに異なる角度で屈折するため、スペクトルに分離される。

導出: プリズムの角度分散

1 最小偏角での条件

頂角 \(A\) のプリズムを最小偏角 \(\delta_m\) で光が通過するとき：

\[ n = \frac{\sin\left(\frac{A + \delta_m}{2}\right)}{\sin\left(\frac{A}{2}\right)} \]
2 角度分散

波長による偏角の変化率（角度分散）：

\[ \frac{d\delta}{d\lambda} = \frac{d\delta}{dn} \cdot \frac{dn}{d\lambda} \]
3 小さな頂角の場合

\(A\) が小さいとき、\(\delta \approx (n-1)A\) より：

\[ \frac{d\delta}{d\lambda} \approx A \frac{dn}{d\lambda} \]

2.6.3 虹の形成

虹は太陽光が水滴によって屈折・反射・分散されて生じる。

  • 主虹（一次の虹）：水滴内で1回反射。外側が赤、内側が紫。視半径約42°
  • 副虹（二次の虹）：水滴内で2回反射。色の順序が逆。視半径約51°
  • アレキサンダーの暗帯：主虹と副虹の間の暗い領域

例題2.5: 虹の視半径

問題: 水の屈折率を n = 1.33 として、主虹の赤色光（n = 1.331）と紫色光（n = 1.344）の視半径の差を計算せよ。

解答:

主虹の視半径 θ は以下の式で与えられる：

\[ \cos\theta = \frac{n^2 - 1}{3} \cdot \sqrt{\frac{3}{4 - n^2}} \]

この式を数値計算すると：

  • 赤色光 (n = 1.331): θ ≈ 42.5°
  • 紫色光 (n = 1.344): θ ≈ 40.5°

視半径の差: 約2°

これが虹の幅（約2°）に対応する。

コラム: 色収差と光学設計

分散は光学機器において「色収差」という問題を引き起こす。単レンズでは波長ごとに焦点位置が異なるため、像がぼやける。

対策として：

  • アクロマートレンズ：屈折率と分散が異なる2種類のガラス（クラウンガラスとフリントガラス）を組み合わせ、2波長で色収差を補正
  • アポクロマートレンズ：3波長で補正。より高性能
  • 反射望遠鏡：反射では分散が起こらないため、色収差がない

演習問題

問題2.1 基礎

黄色い紙に青色光を照射すると何色に見えるか。理由とともに説明せよ。

問題2.2 応用

太陽の高度が30°のとき、天頂方向に比べて大気を通過する距離は何倍か。また、このときレイリー散乱により青色光(λ=450nm)は赤色光(λ=650nm)に比べて何倍減衰しやすいか（減衰係数が λ⁻⁴ に比例すると仮定）。

問題2.3 発展

三色型色覚では、分光器で見ると全く異なるスペクトルを持つ2つの光が同じ色に見えることがある（メタメリズム）。具体例を挙げ、なぜこのような現象が起こるか説明せよ。また、印刷物とディスプレイで同じ色を再現することが難しい理由を色域の観点から説明せよ。

問題2.4 考察

月面から見た「地球の出」では、地球の縁がオレンジ色に見える。この現象を説明せよ。また、火星から見た夕焼けは何色に見えるか予想し、その理由を述べよ（火星大気は薄く、塵が多い）。

第2回のまとめ

学習した主要概念

  • 可視光線は波長約380-780nmの電磁波で、各波長は特定の色として知覚される
  • 人間の色覚は3種類の錐体細胞（L, M, S）に基づく三色説で説明される
  • 物体の色は、吸収されなかった波長の光が反射・透過することで決まる
  • 加法混色（RGB）は光の重ね合わせ、減法混色（CMY）は色素の重ね合わせ
  • レイリー散乱（I ∝ 1/λ⁴）により空は青く、夕焼けは赤く見える
  • 屈折率の波長依存性（分散）によりプリズムで光が分光され、虹が生じる
