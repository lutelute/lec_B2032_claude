電磁波と特殊相対性理論

学習目標: 特殊相対性理論の基本原理を理解し、電磁波との深い関係を説明できるようになる。ローレンツ変換、時間の遅れ、長さの収縮、相対論的ドップラー効果を数学的に導出し、実験的検証と応用例を学ぶ。光速不変の原理がマクスウェル方程式から自然に導かれることを理解する。

12.1 特殊相対性理論の誕生 - エーテルの探求から光速不変へ

12.1.1 19世紀の物理学とエーテル仮説

19世紀末、物理学者たちは電磁波の伝播媒体として「エーテル」という仮想的な物質の存在を信じていた。音波が空気を、水面波が水を媒体として伝播するように、光や電磁波もエーテルという媒体中を伝播すると考えられていた。

エーテル仮説は以下の特性を持つと考えられていた：

  • 宇宙全体に充満：真空中でも光が伝播することの説明
  • 完全に透明：光を吸収しない
  • 極めて剛性が高い：横波（光）の高速伝播を可能にするため
  • 物質と相互作用しない：惑星の運動に影響を与えない
  • 静止している：絶対静止系を定義する基準

このエーテル仮説は多くの矛盾を孕んでいた。剛性が高いにもかかわらず物質と相互作用しないという性質は物理的に不自然であった。

12.1.2 マイケルソン-モーリーの実験（1887年）

地球はエーテルの海の中を約30 km/sの速度で公転している。もしエーテルが存在するなら、地球の運動方向に対して平行・垂直に進む光の速度に差が生じるはずである。アルバート・マイケルソンとエドワード・モーリーは、精密な干渉計を用いてこの速度差の検出を試みた。

マイケルソン干渉計の原理

地球の速度を \(v\)、光速を \(c\)、干渉計の腕の長さを \(L\) とする。

エーテルの風に平行な光路の往復時間：

\[ t_{\parallel} = \frac{L}{c-v} + \frac{L}{c+v} = \frac{2Lc}{c^2 - v^2} = \frac{2L/c}{1 - v^2/c^2} \]

エーテルの風に垂直な光路の往復時間：

\[ t_{\perp} = \frac{2L}{\sqrt{c^2 - v^2}} = \frac{2L/c}{\sqrt{1 - v^2/c^2}} \]

導出: 予想される干渉縞のずれ

1 時間差の計算

\(\beta = v/c \ll 1\) として、テイラー展開を用いる：

\[ t_{\parallel} \approx \frac{2L}{c}\left(1 + \beta^2 + \cdots\right) \] \[ t_{\perp} \approx \frac{2L}{c}\left(1 + \frac{\beta^2}{2} + \cdots\right) \]
2 時間差 \[ \Delta t = t_{\parallel} - t_{\perp} \approx \frac{2L}{c} \cdot \frac{\beta^2}{2} = \frac{Lv^2}{c^3} \]
3 光路差と干渉縞のずれ

光路差: \(\Delta \ell = c \Delta t = Lv^2/c^2\)

干渉縞のずれ: \(\Delta N = \Delta \ell / \lambda\)

4 数値計算

\(L = 11\) m、\(v = 30\) km/s、\(\lambda = 500\) nm として：

\[ \Delta N = \frac{11 \times (3 \times 10^4)^2}{(3 \times 10^8)^2 \times 5 \times 10^{-7}} \approx 0.22 \]

約0.2縞のずれが予想されたが、検出限界（0.01縞程度）をはるかに上回る精度で、ずれは観測されなかった。

コラム: マイケルソン-モーリー実験の衝撃

この「否定的な結果」は物理学界に大きな衝撃を与えた。エーテルを救うために様々な仮説が提唱された：

  • エーテル引きずり仮説：地球がエーテルを一緒に引きずっている（しかし、光行差の観測と矛盾）
  • フィッツジェラルド-ローレンツ収縮：運動方向に物体が収縮する（アドホックな仮説）
  • 発射説：光源の速度が光速に加算される（連星の観測と矛盾）

これらの仮説はすべて何らかの問題を抱えていた。エーテルの存在を前提とする限り、矛盾のない理論を構築することは不可能であった。この状況を根本的に解決したのが、1905年のアインシュタインによる特殊相対性理論である。

12.1.3 アインシュタインの着眼点

アインシュタインは、エーテルの存在を否定するのではなく、より根本的な問いを立てた。「なぜ光の速度は常に一定なのか？」という問いではなく、「光速が一定であることを原理として受け入れたとき、物理法則はどのような形になるべきか？」という問いである。

マクスウェル方程式を詳細に検討すると、電磁波の速度は：

\[ c = \frac{1}{\sqrt{\mu_0 \epsilon_0}} \]

と表される。ここで \(\mu_0\) と \(\epsilon_0\) は真空の透磁率と誘電率であり、観測者の運動状態に依存しない物理定数である。したがって、マクスウェル方程式がすべての慣性系で同じ形を持つなら、光速も同じ値 \(c\) でなければならない。

マクスウェル方程式と光速不変

電磁気学の法則（マクスウェル方程式）は、すべての慣性系で同じ形を持つべきである。この要請から、光速 \(c = 1/\sqrt{\mu_0\epsilon_0}\) がすべての慣性系で同じ値を持つことが自然に導かれる。アインシュタインは、この事実を物理学の基本原理として採用した。

12.2 特殊相対性理論の二つの公準

12.2.1 相対性原理（第一公準）

第一公準：相対性原理

物理法則は、すべての慣性系において同じ形をとる。

数学的表現：物理法則を記述する方程式は、慣性系間の座標変換に対して不変である。

この原理は古典力学のガリレイの相対性原理を拡張したものである。ガリレイの相対性原理は力学の法則についてのみ成立したが、アインシュタインはこれを電磁気学を含むすべての物理法則に拡張した。

相対性原理の物理的意味は以下の通りである：

  • 閉じた実験室内の実験によって、その実験室の一様な直線運動を検出することは不可能である
  • 絶対静止系（エーテル静止系）は存在しない、または物理的に検出不可能である
  • 運動は相対的なものであり、「絶対速度」に物理的意味はない

12.2.2 光速不変の原理（第二公準）

第二公準：光速不変の原理

真空中の光速度 \(c\) は、光源や観測者の運動状態によらず、すべての慣性系で同じ値を持つ。

\[ c = 299,792,458 \text{ m/s}（定義値） \]

この公準は我々の日常的直観に反する。例えば、光速の90%で移動する宇宙船から前方に光を発射しても、静止した観測者が測定する光の速度は \(c + 0.9c = 1.9c\) ではなく、依然として \(c\) である。

例題12.1: 光速不変の直観に反する性質

問題: 宇宙船Aが地球に対して速度 \(0.6c\) で飛行している。宇宙船Aから前方に光を発射した。この光の速度を(1)宇宙船から見た場合、(2)地球から見た場合、それぞれ求めよ。

解答:

(1) 宇宙船から見た光の速度：\(c = 3.0 \times 10^8\) m/s

(2) 地球から見た光の速度：\(c = 3.0 \times 10^8\) m/s

解説: 光速不変の原理により、光の速度はどの慣性系から観測しても常に \(c\) である。日常的な速度の足し算（ガリレイ変換）は光には適用されない。これは特殊相対性理論の根本的な帰結である。

12.2.3 二つの公準から導かれる帰結

この二つの公準を同時に満たすためには、我々の時空間に関する常識を根本的に改める必要がある。主な帰結として：

  • 同時性の相対性：ある慣性系で同時に起こる2つの事象が、別の慣性系では同時でない可能性がある
  • 時間の遅れ：運動する時計は静止した時計よりゆっくり進む
  • 長さの収縮：運動する物体は進行方向に収縮して見える
  • 質量とエネルギーの等価性：\(E = mc^2\)
  • 速度の合成則の修正：光速を超える相対速度は存在しない

12.3 ローレンツ変換

12.3.1 ガリレイ変換の限界

古典力学では、静止系Sと速度 \(v\) で移動する系S'の間の座標変換はガリレイ変換で記述される：

ガリレイ変換
\[ x' = x - vt \] \[ y' = y \] \[ z' = z \] \[ t' = t \]

時間は絶対的であり、すべての慣性系で同じように流れる。

しかし、ガリレイ変換の下ではマクスウェル方程式が不変でない。また、光速不変の原理とも矛盾する。光速不変を満たす座標変換がローレンツ変換である。

12.3.2 ローレンツ変換の導出

導出: ローレンツ変換

1 仮定と設定

静止系Sと、S系に対して \(x\) 軸正方向に速度 \(v\) で運動する系S'を考える。時刻 \(t = t' = 0\) で両系の原点が一致するとする。

変換は線形であると仮定する：

\[ x' = \gamma(x - vt) \] \[ t' = \gamma(t - \alpha x) \]

ここで \(\gamma\) と \(\alpha\) は未知の定数。

2 逆変換の要請

相対性原理より、S'から見たSの速度は \(-v\) である。したがって逆変換は：

\[ x = \gamma(x' + vt') \] \[ t = \gamma(t' + \alpha x') \]
3 光速不変の条件

S系の原点 \((x=0, t=0)\) から発した光が、S系で \((x, t)\) に到達したとする。光速不変より：

\[ x = ct \quad \text{(S系)} \] \[ x' = ct' \quad \text{(S'系)} \]
4 \(\alpha\) の決定

変換式に \(x = ct\) を代入：

\[ x' = \gamma(ct - vt) = \gamma t(c - v) \] \[ t' = \gamma(t - \alpha ct) = \gamma t(1 - \alpha c) \]

\(x' = ct'\) より：

\[ \gamma t(c - v) = c \cdot \gamma t(1 - \alpha c) \] \[ c - v = c - \alpha c^2 \] \[ \alpha = \frac{v}{c^2} \]
5 \(\gamma\) の決定

変換と逆変換を組み合わせる。\(x' = \gamma(x - vt)\) を \(x = \gamma(x' + vt')\) に代入：

\[ x = \gamma\left[\gamma(x - vt) + v\gamma\left(t - \frac{vx}{c^2}\right)\right] \] \[ x = \gamma^2 x - \gamma^2 vt + \gamma^2 vt - \gamma^2\frac{v^2 x}{c^2} \] \[ x = \gamma^2 x\left(1 - \frac{v^2}{c^2}\right) \]

恒等的に成り立つためには：

\[ \gamma^2 = \frac{1}{1 - v^2/c^2} \] \[ \gamma = \frac{1}{\sqrt{1 - v^2/c^2}} = \frac{1}{\sqrt{1 - \beta^2}} \]

ここで \(\beta = v/c\) である。

ローレンツ変換
\[ x' = \gamma(x - vt) \] \[ y' = y \] \[ z' = z \] \[ t' = \gamma\left(t - \frac{vx}{c^2}\right) \]

ここで、ローレンツ因子 \(\gamma\)（ガンマ因子）は：

\[ \gamma = \frac{1}{\sqrt{1 - v^2/c^2}} = \frac{1}{\sqrt{1 - \beta^2}} \]

12.3.3 ローレンツ因子の性質

ローレンツ因子 \(\gamma\) は特殊相対性理論の中心的な量である：

速度 \(v\) \(\beta = v/c\) \(\gamma\) 備考
0 0 1.000 静止状態、古典力学の極限
30 km/s（地球公転） 0.0001 1.000000005 相対論的効果は極めて小さい
\(0.1c\) 0.1 1.005 0.5%の補正
\(0.5c\) 0.5 1.155 15%の効果
\(0.8c\) 0.8 1.667 67%の効果
\(0.9c\) 0.9 2.294 時間が2倍以上遅れる
\(0.99c\) 0.99 7.089 高エネルギー粒子の領域
\(0.999c\) 0.999 22.37 加速器の粒子
\(c\) 1 \(\infty\) 光速に到達不可能

例題12.2: ローレンツ変換の適用

問題: 静止系Sで、座標 \((x, t) = (10 \text{ m}, 5 \times 10^{-8} \text{ s})\) で起こった事象がある。S系に対して速度 \(0.6c\) で \(x\) 軸正方向に運動する系S'では、この事象はどの座標で観測されるか。

解答:

\(v = 0.6c = 1.8 \times 10^8\) m/s として、まずローレンツ因子を計算する：

\[ \gamma = \frac{1}{\sqrt{1 - 0.6^2}} = \frac{1}{\sqrt{0.64}} = \frac{1}{0.8} = 1.25 \]

座標変換：

\[ x' = \gamma(x - vt) = 1.25\left(10 - 1.8 \times 10^8 \times 5 \times 10^{-8}\right) \] \[ x' = 1.25(10 - 9) = 1.25 \text{ m} \] \[ t' = \gamma\left(t - \frac{vx}{c^2}\right) = 1.25\left(5 \times 10^{-8} - \frac{1.8 \times 10^8 \times 10}{(3 \times 10^8)^2}\right) \] \[ t' = 1.25\left(5 \times 10^{-8} - 2 \times 10^{-8}\right) = 3.75 \times 10^{-8} \text{ s} \]

S'系での座標は \((x', t') = (1.25 \text{ m}, 37.5 \text{ ns})\) となる。

12.4 時間の遅れと長さの収縮

12.4.1 時間の遅れ（時間膨張）

運動する時計は、静止した時計と比べてゆっくり進む。この効果を「時間の遅れ」または「時間膨張」（time dilation）という。

導出: 時間の遅れ

1 設定

S'系で静止している時計を考える。この時計は S'系の原点 \(x' = 0\) に置かれているとする。

2 固有時間

S'系でこの時計が時間間隔 \(\Delta t'\)（固有時間、proper time）を刻んだとする。

3 S系での時間間隔

ローレンツ変換の逆変換 \(t = \gamma(t' + vx'/c^2)\) を用いる。

\(x' = 0\) より：

\[ t = \gamma t' \]

したがって、S系での時間間隔は：

\[ \Delta t = \gamma \Delta t' \]
4 結論

\(\gamma > 1\) より \(\Delta t > \Delta t'\)。S系から見ると、運動する時計はゆっくり進む。

時間の遅れ
\[ \Delta t = \gamma \Delta \tau = \frac{\Delta \tau}{\sqrt{1 - v^2/c^2}} \]

\(\Delta \tau\)：固有時間（時計と一緒に運動する系で測った時間）

\(\Delta t\)：静止系で測った時間

コラム: ミューオンの寿命と時間の遅れ

宇宙線が大気上層で空気分子と衝突すると、ミューオン（μ粒子）が生成される。ミューオンの静止時の寿命は約2.2 μsで、光速で移動しても \(c \times 2.2 \text{ μs} \approx 660\) m しか進めない。

しかし、実際には地表でも多数のミューオンが観測される。大気上層（高度約15 km）で生成されたミューオンが地表に到達するのは、時間の遅れの効果による。

ミューオンが速度 \(0.998c\) で運動する場合：

\[ \gamma = \frac{1}{\sqrt{1 - 0.998^2}} \approx 15.8 \]

地上の観測者から見たミューオンの寿命は：

\[ \Delta t = 15.8 \times 2.2 \text{ μs} \approx 35 \text{ μs} \]

この間にミューオンが進む距離：

\[ d = 0.998c \times 35 \text{ μs} \approx 10.5 \text{ km} \]

これで大気を通過して地表に到達できることが説明される。

12.4.2 長さの収縮（ローレンツ収縮）

運動する物体は、進行方向に収縮して観測される。この効果を「長さの収縮」または「ローレンツ収縮」という。

導出: 長さの収縮

1 設定

S'系で静止している棒を考える。S'系で測った棒の長さを \(L_0\)（固有長）とする。棒の両端の座標は \(x'_1 = 0\)、\(x'_2 = L_0\) である。

2 S系での測定

S系で棒の長さを測定するには、ある同時刻（\(t\) 一定）に両端の位置を測定する必要がある。

3 座標変換

ローレンツ変換 \(x' = \gamma(x - vt)\) より、時刻 \(t\) における両端の位置は：

\[ x'_1 = \gamma(x_1 - vt), \quad x'_2 = \gamma(x_2 - vt) \] \[ L_0 = x'_2 - x'_1 = \gamma(x_2 - x_1) = \gamma L \]
4 結論 \[ L = \frac{L_0}{\gamma} = L_0\sqrt{1 - v^2/c^2} \]

\(\gamma > 1\) より \(L < L_0\)。運動する棒は静止時より短く見える。

長さの収縮
\[ L = \frac{L_0}{\gamma} = L_0\sqrt{1 - v^2/c^2} \]

\(L_0\)：固有長（物体と一緒に運動する系で測った長さ）

\(L\)：静止系で測った長さ

注意：収縮は運動方向のみで起こり、垂直方向の長さは変化しない。

例題12.3: 双子のパラドックス

問題: 地球に残る双子Aと、速度 \(0.8c\) で10光年先の星まで往復する双子Bがいる。往復に要する時間は、Aの時計とBの時計でそれぞれ何年か。

解答:

まず、Aから見た往復時間を計算する。

片道距離 \(d = 10\) 光年、速度 \(v = 0.8c\) より：

\[ t_A = \frac{2d}{v} = \frac{2 \times 10}{0.8} = 25 \text{ 年} \]

次に、ローレンツ因子を計算する：

\[ \gamma = \frac{1}{\sqrt{1 - 0.8^2}} = \frac{1}{\sqrt{0.36}} = \frac{1}{0.6} \approx 1.667 \]

Bの時計（固有時間）は：

\[ t_B = \frac{t_A}{\gamma} = \frac{25}{1.667} = 15 \text{ 年} \]

結論: Aは25歳、Bは15歳だけ年を取る。帰還時、Bは Aより10歳若い。

補足（パラドックスの解消）: これは「パラドックス」と呼ばれるが、実際には矛盾はない。Bは折り返し点で加速（減速・反転）を経験するため、AとBは対称ではない。Bは非慣性系を経験し、この非対称性がBの方が若くなる物理的理由である。

12.4.3 ミューオンから見た長さの収縮

ミューオンの寿命の問題は、長さの収縮の観点からも説明できる。

例題12.4: ミューオンの視点

問題: 速度 \(0.998c\) で移動するミューオンの視点から、大気の厚さ（15 km）はどのように見えるか。

解答:

ローレンツ因子：\(\gamma = 15.8\)

ミューオンから見た大気の厚さ：

\[ L = \frac{L_0}{\gamma} = \frac{15 \text{ km}}{15.8} \approx 0.95 \text{ km} = 950 \text{ m} \]

ミューオンから見ると、大気はわずか950 mに収縮している。

ミューオンの静止時寿命 2.2 μs の間に光速で進める距離：

\[ d = c \times 2.2 \text{ μs} = 660 \text{ m} \]

0.998c での移動距離：

\[ d = 0.998c \times 2.2 \text{ μs} \approx 660 \text{ m} \]

これでもまだ950 mより短いが、確率的に一部のミューオンは地表に到達できる（寿命の統計的分布を考慮）。

注意: 地上から見た「時間の遅れ」と、ミューオンから見た「長さの収縮」は、同じ物理現象を異なる視点から見たものであり、両者は整合している。

12.5 相対論的速度の合成

12.5.1 ガリレイ変換での速度合成

古典力学では、速度の合成は単純な足し算である。地面に対して速度 \(v\) で走る電車の中で、進行方向に速度 \(u'\) でボールを投げると、地面から見たボールの速度は \(u = u' + v\) となる。

しかし、この単純な合成則は光速不変の原理と矛盾する。もし \(u' = c\) なら、\(u = c + v > c\) となってしまう。

12.5.2 相対論的速度合成則の導出

導出: 相対論的速度合成則

1 設定

S系に対して速度 \(v\) で運動するS'系を考える。S'系で物体が速度 \(u'_x\) で運動しているとする。

2 速度の定義

S系での速度：\(u_x = dx/dt\)

S'系での速度：\(u'_x = dx'/dt'\)

3 ローレンツ変換の微分

\(x = \gamma(x' + vt')\) を微分：

\[ dx = \gamma(dx' + v\,dt') \]

\(t = \gamma(t' + vx'/c^2)\) を微分：

\[ dt = \gamma\left(dt' + \frac{v}{c^2}dx'\right) \]
4 速度の計算 \[ u_x = \frac{dx}{dt} = \frac{\gamma(dx' + v\,dt')}{\gamma(dt' + \frac{v}{c^2}dx')} = \frac{dx'/dt' + v}{1 + \frac{v}{c^2}dx'/dt'} \] \[ u_x = \frac{u'_x + v}{1 + \frac{u'_x v}{c^2}} \]
相対論的速度合成則
\[ u_x = \frac{u'_x + v}{1 + \frac{u'_x v}{c^2}} \]

3次元の場合（運動方向に垂直な成分）：

\[ u_y = \frac{u'_y}{\gamma\left(1 + \frac{u'_x v}{c^2}\right)}, \quad u_z = \frac{u'_z}{\gamma\left(1 + \frac{u'_x v}{c^2}\right)} \]

12.5.3 速度合成則の性質

相対論的速度合成則には重要な性質がある：

  • 低速極限：\(u', v \ll c\) のとき、分母 \(\approx 1\) となり、\(u \approx u' + v\)（ガリレイ変換に帰着）
  • 光速の不変性：\(u' = c\) のとき、\(v\) がいくらでも \(u = c\) となる
  • 光速未満の保存：\(u' < c\) かつ \(v < c\) なら、常に \(u < c\) となる

例題12.5: 相対論的速度合成

問題: 地球に対して速度 \(0.9c\) で飛行する宇宙船から、進行方向に速度 \(0.9c\) でロケットを発射した。地球から見たロケットの速度はいくらか。

解答:

古典力学なら：\(u = 0.9c + 0.9c = 1.8c\)（光速を超えてしまう！）

相対論的速度合成則を用いる：

\[ u = \frac{u' + v}{1 + \frac{u' v}{c^2}} = \frac{0.9c + 0.9c}{1 + \frac{0.9c \times 0.9c}{c^2}} \] \[ u = \frac{1.8c}{1 + 0.81} = \frac{1.8c}{1.81} \approx 0.994c \]

ロケットの速度は約 \(0.994c\) であり、光速を超えない。

例題12.6: 光の速度合成

問題: 速度 \(0.9c\) で飛行する宇宙船から、進行方向に光（速度 \(c\)）を発射した。地球から見た光の速度はいくらか。

解答:

\[ u = \frac{c + 0.9c}{1 + \frac{c \times 0.9c}{c^2}} = \frac{1.9c}{1 + 0.9} = \frac{1.9c}{1.9} = c \]

光の速度は光源の運動によらず常に \(c\) である。これは光速不変の原理そのものである。

12.6 相対論的ドップラー効果

12.6.1 古典的ドップラー効果

音波などの古典的な波では、ドップラー効果は光源と観測者の媒質に対する運動によって決まる。光源が速度 \(v_s\) で観測者に近づき、観測者が速度 \(v_o\) で光源に近づく場合：

\[ f' = f \cdot \frac{c + v_o}{c - v_s} \]

しかし、光には媒質がないため、光源と観測者の相対速度のみが意味を持つ。

12.6.2 相対論的ドップラー効果の導出

導出: 縦ドップラー効果（光源が視線方向に運動）

1 光源の固有周期

光源が静止系で周期 \(T_0\) で発光するとする。

2 時間の遅れの効果

観測者から見ると、運動する光源の時計はゆっくり進む：

\[ T_{source} = \gamma T_0 \]
3 古典的なドップラー効果

光源が観測者から遠ざかる場合、連続する波面の間隔（波長）が長くなる：

\[ \lambda' = \lambda + v T_{source} = c T_{source} + v T_{source} = (c + v) T_{source} \]
4 観測される周波数 \[ f' = \frac{c}{\lambda'} = \frac{c}{(c + v) T_{source}} = \frac{c}{(c + v) \gamma T_0} \] \[ f' = \frac{f_0}{\gamma(1 + v/c)} = \frac{f_0 \sqrt{1 - v^2/c^2}}{1 + v/c} \] \[ f' = f_0 \sqrt{\frac{1 - v/c}{1 + v/c}} \]
相対論的ドップラー効果

光源が遠ざかる場合（赤方偏移）:

\[ f_{obs} = f_0 \sqrt{\frac{1 - \beta}{1 + \beta}} = f_0 \sqrt{\frac{c - v}{c + v}} \]

光源が近づく場合（青方偏移）:

\[ f_{obs} = f_0 \sqrt{\frac{1 + \beta}{1 - \beta}} = f_0 \sqrt{\frac{c + v}{c - v}} \]

ここで \(\beta = v/c\)、\(v\) は遠ざかる速度を正とする。

12.6.3 横ドップラー効果

特殊相対性理論では、光源が観測者に対して垂直方向に運動する場合でもドップラー効果が生じる。これは古典力学にはない純粋に相対論的な効果であり、時間の遅れに起因する。

横ドップラー効果
\[ f_{obs} = \frac{f_0}{\gamma} = f_0 \sqrt{1 - v^2/c^2} \]

運動する光源から垂直方向に放射された光は、常に赤方偏移する。

例題12.7: 宇宙論的赤方偏移

問題: 遠方の銀河から放射された水素のHα線（静止波長656.3 nm）が、波長759 nmで観測された。この銀河の後退速度を求めよ。

解答:

赤方偏移パラメータ \(z\) を定義する：

\[ z = \frac{\lambda_{obs} - \lambda_0}{\lambda_0} = \frac{759 - 656.3}{656.3} = 0.156 \]

相対論的ドップラー公式より：

\[ 1 + z = \frac{\lambda_{obs}}{\lambda_0} = \sqrt{\frac{1 + \beta}{1 - \beta}} \] \[ (1 + z)^2 = \frac{1 + \beta}{1 - \beta} \] \[ \beta = \frac{(1 + z)^2 - 1}{(1 + z)^2 + 1} = \frac{1.156^2 - 1}{1.156^2 + 1} = \frac{0.337}{2.337} = 0.144 \]

後退速度：\(v = 0.144c \approx 4.3 \times 10^7\) m/s \(\approx 43,000\) km/s

コラム: GPSと相対論的効果

全地球測位システム（GPS）は、特殊相対性理論と一般相対性理論の両方の効果を補正しなければ正確に機能しない。

特殊相対論的効果（時間の遅れ）:

GPS衛星は約4 km/sで地球を周回している。この速度での時間の遅れは：

\[ \Delta t_{SR} = t_0 \left(1 - \sqrt{1 - v^2/c^2}\right) \approx t_0 \cdot \frac{v^2}{2c^2} \]

1日あたり約7 μsの遅れが生じる。

一般相対論的効果（重力による時間の遅れ）:

衛星は地表より弱い重力場にあるため、衛星の時計は地上より速く進む。この効果は1日あたり約45 μsの進みをもたらす。

正味の効果:

合計で1日あたり約38 μsの補正が必要。これを補正しないと、1日で約11 kmの位置誤差が累積する。

12.7 質量とエネルギーの等価性

12.7.1 相対論的運動量

古典力学での運動量 \(p = mv\) は、相対論では修正が必要である。運動量保存則がすべての慣性系で成立するためには：

相対論的運動量
\[ \vec{p} = \gamma m \vec{v} = \frac{m\vec{v}}{\sqrt{1 - v^2/c^2}} \]

\(m\) は静止質量（不変質量）

12.7.2 相対論的エネルギー

導出: 相対論的運動エネルギー

1 仕事と運動エネルギーの関係

力 \(F\) が物体に行う仕事は運動エネルギーの変化に等しい：

\[ dE = F \, dx = \frac{dp}{dt} \cdot v \, dt = v \, dp \]
2 \(dp\) の計算

\(p = \gamma mv\) より：

\[ dp = m \, d(\gamma v) = m\left(\gamma \, dv + v \, d\gamma\right) \]

\(\gamma = (1 - v^2/c^2)^{-1/2}\) を \(v\) で微分：

\[ d\gamma = \frac{v/c^2}{(1 - v^2/c^2)^{3/2}} dv = \frac{\gamma^3 v}{c^2} dv \]
3 エネルギーの積分 \[ dE = v \cdot m\left(\gamma + \frac{\gamma^3 v^2}{c^2}\right) dv = m\gamma v \left(1 + \frac{\gamma^2 v^2}{c^2}\right) dv \]

\(1 + \gamma^2 v^2/c^2 = \gamma^2\) を用いると：

\[ dE = m\gamma^3 v \, dv = mc^2 \, d\gamma \]
4 積分

\(v = 0\)（\(\gamma = 1\)）から \(v\)（\(\gamma\)）まで積分：

\[ E - E_0 = mc^2(\gamma - 1) \]

全エネルギー：\(E = \gamma mc^2 = E_0 + K\)

ここで \(E_0 = mc^2\) は静止エネルギー、\(K = (\gamma - 1)mc^2\) は運動エネルギー。

質量とエネルギーの等価性

静止エネルギー:

\[ E_0 = mc^2 \]

全エネルギー:

\[ E = \gamma mc^2 = \frac{mc^2}{\sqrt{1 - v^2/c^2}} \]

運動エネルギー:

\[ K = E - E_0 = (\gamma - 1)mc^2 \]

エネルギー・運動量関係:

\[ E^2 = (pc)^2 + (mc^2)^2 \]

12.7.3 光子のエネルギーと運動量

質量ゼロの粒子（光子）の場合、\(m = 0\) なので：

\[ E = pc \]

これはプランクの関係式 \(E = hf\) とド・ブロイの関係式 \(p = h/\lambda\) と整合する。

例題12.8: 電子の相対論的エネルギー

問題: 電子（静止質量 \(m_e = 9.11 \times 10^{-31}\) kg）が速度 \(0.99c\) で運動しているとき、(1) 静止エネルギー、(2) 全エネルギー、(3) 運動エネルギー、(4) 運動量を求めよ。

解答:

(1) 静止エネルギー：

\[ E_0 = m_e c^2 = 9.11 \times 10^{-31} \times (3 \times 10^8)^2 \] \[ E_0 = 8.2 \times 10^{-14} \text{ J} = 0.511 \text{ MeV} \]

(2) ローレンツ因子：

\[ \gamma = \frac{1}{\sqrt{1 - 0.99^2}} = \frac{1}{\sqrt{0.0199}} = 7.09 \]

全エネルギー：

\[ E = \gamma m_e c^2 = 7.09 \times 0.511 = 3.62 \text{ MeV} \]

(3) 運動エネルギー：

\[ K = E - E_0 = 3.62 - 0.511 = 3.11 \text{ MeV} \]

(4) 運動量：

\[ p = \gamma m_e v = 7.09 \times 9.11 \times 10^{-31} \times 0.99 \times 3 \times 10^8 \] \[ p = 1.92 \times 10^{-21} \text{ kg·m/s} = 3.58 \text{ MeV}/c \]

古典力学との比較:

古典的運動エネルギー \(K_{classical} = \frac{1}{2}mv^2 = \frac{1}{2} \times 0.511 \times 0.99^2 = 0.25\) MeV

相対論的効果により、運動エネルギーは古典値の約12倍になる。

例題12.9: 核融合のエネルギー

問題: 太陽の中心で起こる陽子-陽子連鎖反応では、4つの水素原子核（陽子）から1つのヘリウム4原子核が生成される。質量欠損からこの反応で放出されるエネルギーを計算せよ。

データ: 陽子の質量 \(m_p = 1.00728\) u、ヘリウム4の質量 \(m_{He} = 4.00260\) u、1 u = 931.5 MeV/c²

解答:

反応前の質量（陽子4個）：

\[ M_{before} = 4 \times 1.00728 = 4.02912 \text{ u} \]

反応後の質量（ヘリウム4）：

\[ M_{after} = 4.00260 \text{ u} \]

質量欠損：

\[ \Delta m = M_{before} - M_{after} = 4.02912 - 4.00260 = 0.02652 \text{ u} \]

放出エネルギー：

\[ E = \Delta m \cdot c^2 = 0.02652 \times 931.5 = 24.7 \text{ MeV} \]

（実際には2個の陽電子と2個のニュートリノも放出され、正味のエネルギーは約26.7 MeV）

効率:

\[ \eta = \frac{\Delta m}{M_{before}} = \frac{0.02652}{4.02912} = 0.66\% \]

質量の約0.66%がエネルギーに変換される。化学反応（約10⁻⁸%）と比べて桁違いに効率が高い。

12.8 電磁場のローレンツ変換

12.8.1 電場と磁場の相対性

特殊相対性理論の重要な帰結の一つは、電場と磁場が観測者の運動状態によって混合されることである。ある慣性系で純粋な電場として観測される場が、別の慣性系では電場と磁場の両方として観測される。

電磁場のローレンツ変換

S系からS'系（\(x\)方向に速度 \(v\) で運動）への変換：

運動方向に平行な成分:

\[ E'_x = E_x, \quad B'_x = B_x \]

運動方向に垂直な成分:

\[ E'_y = \gamma(E_y - vB_z), \quad E'_z = \gamma(E_z + vB_y) \] \[ B'_y = \gamma\left(B_y + \frac{v}{c^2}E_z\right), \quad B'_z = \gamma\left(B_z - \frac{v}{c^2}E_y\right) \]

12.8.2 電磁場の不変量

電磁場には、ローレンツ変換で不変に保たれる量がある：

電磁場のローレンツ不変量
\[ \vec{E} \cdot \vec{B} = \text{不変} \] \[ E^2 - c^2 B^2 = \text{不変} \]

これらの不変量から、以下のことが言える：

  • もしある慣性系で \(\vec{E} \perp \vec{B}\) なら、すべての慣性系で \(\vec{E} \perp \vec{B}\)
  • もしある慣性系で \(E > cB\) なら、\(\vec{B} = 0\) となる慣性系が存在する
  • もしある慣性系で \(E < cB\) なら、\(\vec{E} = 0\) となる慣性系が存在する

例題12.10: 運動する点電荷の磁場

問題: 静止した点電荷は周囲に電場のみを作り、磁場は作らない。この点電荷が速度 \(v\) で運動するとき、磁場が生じることを電磁場のローレンツ変換から説明せよ。

解答:

点電荷が静止しているS'系では、\(\vec{B}' = 0\) であり、電場のみが存在する：

\[ \vec{E}' = \frac{q}{4\pi\epsilon_0 r'^2} \hat{r}' \]

点電荷が速度 \(v\) で \(x\) 方向に運動するS系での電磁場を求める。

逆変換より（\(B'_y = B'_z = 0\) を用いて）：

\[ B_y = \gamma\left(B'_y - \frac{v}{c^2}E'_z\right) = -\frac{\gamma v}{c^2}E'_z \] \[ B_z = \gamma\left(B'_z + \frac{v}{c^2}E'_y\right) = \frac{\gamma v}{c^2}E'_y \]

ベクトル形式で書くと：

\[ \vec{B} = \frac{\gamma v}{c^2}(\hat{x} \times \vec{E}'_\perp) = \frac{1}{c^2}\vec{v} \times \vec{E}_\perp \]

これはビオ・サバールの法則から導かれる運動する点電荷の磁場：

\[ \vec{B} = \frac{\mu_0 q}{4\pi} \frac{\vec{v} \times \hat{r}}{r^2} \]

と整合する（\(\mu_0 \epsilon_0 = 1/c^2\) を用いる）。

結論: 磁場は電場の相対論的効果として理解できる。静止した電荷の電場を運動する観測者から見ると、磁場成分が現れる。

コラム: アインシュタインと電磁気学

アインシュタインが特殊相対性理論を構築するきっかけの一つは、電磁気学における非対称性への疑問であった。

磁石とコイルの相対運動を考える：

  • 磁石が静止し、コイルが動く場合：コイル内の電子に働くローレンツ力により電流が流れる
  • コイルが静止し、磁石が動く場合：変化する磁場がファラデーの法則により起電力を生じ、電流が流れる

物理的には同じ現象（相対運動による電流の発生）なのに、説明のメカニズムが異なるのは不自然である。アインシュタインは、この非対称性は見かけのものであり、電場と磁場は観測者によって混合される同じ実体の異なる側面であると考えた。

1905年の論文「運動物体の電気力学について」は、まさにこの問題意識から出発している。

12.9 福井大学遠赤外領域開発研究センターと相対論的電子ビーム

12.9.1 相対論的電子ビームと電磁波発生

福井大学遠赤外領域開発研究センター（FIR-UF）では、ジャイロトロンや自由電子レーザーなど、相対論的電子ビームを用いた電磁波発生装置の研究開発を行っている。これらの装置では、特殊相対性理論が本質的な役割を果たす。

相対論的効果の重要性

ジャイロトロンでは、電子が磁場中で円運動（サイクロトロン運動）しながら電磁波を放射する。電子の速度が光速に近づくと、以下の相対論的効果が重要になる：

  • 相対論的質量増加: 電子のサイクロトロン周波数が低下する
  • 相対論的ドップラー効果: 放射周波数が変調される
  • 相対論的収束: 放射の指向性が高まる

12.9.2 サイクロトロン放射と相対論

磁場中を運動する電子のサイクロトロン周波数は：

相対論的サイクロトロン周波数

非相対論的:

\[ f_c = \frac{eB}{2\pi m_e} \]

相対論的:

\[ f_c = \frac{eB}{2\pi \gamma m_e} \]

相対論的効果により、高速電子のサイクロトロン周波数は低下する。

ジャイロトロンでは、この相対論的効果を利用して、特定の周波数で強い電磁波放射を実現している。電子と電磁波の相互作用において、エネルギーを失った電子はサイクロトロン周波数が増加し、逆にエネルギーを得た電子は周波数が低下する。この「位相集群効果」により、効率的なエネルギー変換が可能となる。

例題12.11: ジャイロトロンの動作周波数

問題: 磁場強度 \(B = 14\) T のジャイロトロンにおいて、電子の加速電圧が 80 kV のとき、基本波の放射周波数を求めよ。

解答:

まず、電子のローレンツ因子を計算する。電子の運動エネルギーは：

\[ K = eV = 80 \text{ keV} \]

電子の静止エネルギーは 511 keV なので：

\[ \gamma = 1 + \frac{K}{m_e c^2} = 1 + \frac{80}{511} = 1.157 \]

相対論的サイクロトロン周波数：

\[ f_c = \frac{eB}{2\pi \gamma m_e} = \frac{1.6 \times 10^{-19} \times 14}{2\pi \times 1.157 \times 9.11 \times 10^{-31}} \] \[ f_c = \frac{2.24 \times 10^{-18}}{6.62 \times 10^{-30}} = 338 \text{ GHz} \]

非相対論的な値 \(f_c = eB/(2\pi m_e) = 392\) GHz と比較すると、相対論的効果により約14%低下している。

12.9.3 自由電子レーザーと相対論

自由電子レーザー（FEL）では、相対論的電子ビームが周期的磁場（アンジュレータ）中を通過する際に電磁波を放射する。放射波長は：

自由電子レーザーの波長
\[ \lambda = \frac{\lambda_u}{2\gamma^2}\left(1 + \frac{K^2}{2}\right) \]

\(\lambda_u\): アンジュレータ周期、\(K\): アンジュレータパラメータ

\(\gamma^{-2}\) の因子は二重の相対論的効果（長さ収縮とドップラー効果）に由来する。

福井大学では、テラヘルツ領域の自由電子レーザー開発も進められており、相対論的電子ビーム技術が重要な役割を果たしている。

演習問題

問題12.1 基礎

宇宙船が地球に対して速度 \(0.8c\) で飛行している。宇宙船内の時計で1時間が経過したとき、地球の時計では何時間経過しているか。

問題12.2 基礎

静止時の長さが100 mの宇宙船が、速度 \(0.6c\) で地球の近くを通過した。地球の観測者から見た宇宙船の長さはいくらか。

問題12.3 応用

地球に対して速度 \(0.5c\) で飛行する宇宙船Aから、進行方向に速度 \(0.5c\) で宇宙船Bが発進した。地球から見た宇宙船Bの速度を求めよ。また、宇宙船Aから見て後方に速度 \(0.5c\) でミサイルを発射した場合、地球から見たミサイルの速度を求めよ。

問題12.4 応用

遠方の銀河から放射されたスペクトル線が、静止波長の2倍の波長で観測された（\(z = 1\)）。この銀河の後退速度を光速の何倍か求めよ。

問題12.5 発展

陽子（静止質量 938.3 MeV/c²）が光速の 99.9% で運動している。(1) ローレンツ因子、(2) 全エネルギー、(3) 運動エネルギー、(4) 運動量を求めよ。

問題12.6 発展

2つの宇宙船が地球からそれぞれ反対方向に速度 \(0.9c\) で飛び去った。一方の宇宙船から見た他方の宇宙船の速度を求めよ。

問題12.7 考察

マイケルソン-モーリー実験の「null結果」を、以下の2つの立場から説明せよ：(1) エーテル引きずり仮説、(2) 特殊相対性理論。それぞれの説明の問題点または利点を述べよ。

問題12.8 発展

一様な電場 \(\vec{E} = E\hat{y}\) のみが存在する系（磁場 \(\vec{B} = 0\)）を、\(x\) 方向に速度 \(v\) で運動する観測者から見ると、どのような電磁場が観測されるか。電磁場のローレンツ変換を用いて計算せよ。

第12回のまとめ

学習した主要概念

  • マイケルソン-モーリー実験はエーテルの存在を否定し、特殊相対性理論誕生のきっかけとなった
  • 特殊相対性理論は相対性原理と光速不変の原理という2つの公準に基づく
  • ローレンツ変換は光速不変を満たす座標変換であり、ローレンツ因子 \(\gamma = 1/\sqrt{1-v^2/c^2}\) が重要な役割を果たす
  • 時間の遅れ（運動する時計はゆっくり進む）と長さの収縮（運動する物体は進行方向に縮む）は相対論の直接的帰結
  • 相対論的速度合成則により、光速を超える相対速度は存在しない
  • 相対論的ドップラー効果は天文学や GPS など多くの応用がある
  • 質量とエネルギーは等価であり、\(E = mc^2\) で結ばれる
  • 電場と磁場は観測者の運動状態によって混合され、本質的に同じ実体の異なる側面である

重要公式

ローレンツ因子: \(\gamma = \frac{1}{\sqrt{1 - v^2/c^2}}\)

時間の遅れ: \(\Delta t = \gamma \Delta \tau\)

長さの収縮: \(L = L_0/\gamma\)

速度合成: \(u = \frac{u' + v}{1 + u'v/c^2}\)

ドップラー効果: \(f_{obs} = f_0\sqrt{\frac{1 \mp \beta}{1 \pm \beta}}\)

質量エネルギー: \(E = \gamma mc^2\)、\(E^2 = (pc)^2 + (mc^2)^2\)
