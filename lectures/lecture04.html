電磁波と物質との相互作用

学習目標: 電磁波が物質中を伝播する際の様々な相互作用を理解する。反射、屈折、吸収、散乱の物理的メカニズムを学び、物質の誘電率と導電率が電磁波応答をどのように決定するかを理解する。光電効果、コンプトン散乱など量子効果についても学習する。

4.1 物質中の電磁波

4.1.1 マクスウェル方程式と物質

真空中のマクスウェル方程式は物質中では修正される。物質の電磁気的性質は、誘電率 \(\epsilon\)、透磁率 \(\mu\)、導電率 \(\sigma\) によって特徴づけられる。

物質中のマクスウェル方程式
\[ \nabla \cdot \vec{D} = \rho_f \] \[ \nabla \cdot \vec{B} = 0 \] \[ \nabla \times \vec{E} = -\frac{\partial \vec{B}}{\partial t} \] \[ \nabla \times \vec{H} = \vec{J}_f + \frac{\partial \vec{D}}{\partial t} \]

構成方程式: \(\vec{D} = \epsilon \vec{E}\)、\(\vec{B} = \mu \vec{H}\)、\(\vec{J} = \sigma \vec{E}\)

物質中での電磁波の伝播を考えるため、自由電荷・自由電流がない場合（\(\rho_f = 0\)、\(\vec{J}_f = \sigma \vec{E}\)）の波動方程式を導出する。

導出: 物質中の波動方程式

1 ファラデーの法則の回転を取る \[ \nabla \times (\nabla \times \vec{E}) = -\frac{\partial}{\partial t}(\nabla \times \vec{B}) = -\mu \frac{\partial}{\partial t}(\nabla \times \vec{H}) \]
2 アンペール-マクスウェルの法則を代入 \[ \nabla \times (\nabla \times \vec{E}) = -\mu \frac{\partial}{\partial t}\left(\sigma \vec{E} + \epsilon \frac{\partial \vec{E}}{\partial t}\right) \]
3 ベクトル恒等式を使用

\(\nabla \times (\nabla \times \vec{E}) = \nabla(\nabla \cdot \vec{E}) - \nabla^2 \vec{E}\)

\(\nabla \cdot \vec{E} = 0\)（電荷がない場合）より：

\[ -\nabla^2 \vec{E} = -\mu\sigma \frac{\partial \vec{E}}{\partial t} - \mu\epsilon \frac{\partial^2 \vec{E}}{\partial t^2} \]
4 波動方程式 \[ \nabla^2 \vec{E} = \mu\sigma \frac{\partial \vec{E}}{\partial t} + \mu\epsilon \frac{\partial^2 \vec{E}}{\partial t^2} \]

第1項は損失項（導電率による減衰）、第2項は通常の波動項である。

4.1.2 複素屈折率

角周波数 \(\omega\) の平面波 \(\vec{E} = \vec{E}_0 e^{i(\tilde{k}z - \omega t)}\) を仮定すると、波数は複素数となる：

複素波数と複素屈折率
\[ \tilde{k}^2 = \mu\epsilon\omega^2 + i\mu\sigma\omega = \frac{\omega^2}{c^2}\tilde{n}^2 \]

複素屈折率: \(\tilde{n} = n + i\kappa\)

\(n\): 屈折率（実部）、\(\kappa\): 消衰係数

複素波数を \(\tilde{k} = k' + ik''\) とすると、電場は：

\[ \vec{E} = \vec{E}_0 e^{-k''z} e^{i(k'z - \omega t)} \]

第1の因子 \(e^{-k''z}\) は振幅の減衰を表し、第2の因子は位相速度 \(v_p = \omega/k'\) での伝播を表す。

4.1.3 誘電関数

物質の電磁波応答は複素誘電関数 \(\tilde{\epsilon}(\omega)\) で記述できる：

複素誘電関数
\[ \tilde{\epsilon}(\omega) = \epsilon'(\omega) + i\epsilon''(\omega) = \epsilon(\omega) + i\frac{\sigma(\omega)}{\omega} \]

複素屈折率との関係: \(\tilde{n}^2 = \frac{\tilde{\epsilon}}{\epsilon_0}\)（非磁性体で \(\mu = \mu_0\) の場合）

実部 \(\epsilon'\) は電場に対する分極の同相成分（エネルギー蓄積）、虚部 \(\epsilon''\) は直交成分（エネルギー吸収）に関係する。

例題4.1: 複素屈折率から吸収係数へ

問題: ある材料の複素屈折率が \(\tilde{n} = 1.5 + 0.01i\) のとき、波長 500 nm の光に対する吸収係数 \(\alpha\)（強度が 1/e になる距離の逆数）を求めよ。

解答:

電場の減衰: \(E \propto e^{-k''z}\) より、強度の減衰は \(I \propto |E|^2 \propto e^{-2k''z}\)

吸収係数は \(\alpha = 2k''\) で定義される。

\(k'' = \frac{\omega \kappa}{c} = \frac{2\pi \kappa}{\lambda}\) より：

\[ \alpha = 2 \times \frac{2\pi \times 0.01}{500 \times 10^{-9}} = \frac{4\pi \times 0.01}{500 \times 10^{-9}} \] \[ = 2.51 \times 10^5 \text{ m}^{-1} = 2.51 \times 10^3 \text{ cm}^{-1} \]

光の強度は \(1/\alpha \approx 4 \times 10^{-6}\) m = 4 μm で 1/e に減衰する。

4.2 反射と屈折

4.2.1 境界条件

異なる媒質の界面で電磁波が満たすべき境界条件は、マクスウェル方程式から導かれる：

電磁場の境界条件
  • \(E_{1t} = E_{2t}\)：電場の接線成分は連続
  • \(D_{1n} - D_{2n} = \sigma_s\)：電束密度の法線成分の不連続 = 表面電荷密度
  • \(B_{1n} = B_{2n}\)：磁束密度の法線成分は連続
  • \(H_{1t} - H_{2t} = K_s\)：磁場の接線成分の不連続 = 表面電流密度

4.2.2 スネルの法則

電磁波が屈折率 \(n_1\) の媒質から \(n_2\) の媒質に入射するとき、入射角 \(\theta_1\) と屈折角 \(\theta_2\) の関係は位相整合条件から導かれる。

導出: スネルの法則

1 位相整合条件

界面（z = 0）上のすべての点で、入射波・反射波・透過波の位相が一致する必要がある。

2 波数ベクトルの界面平行成分

入射波: \(k_{1x} = k_1 \sin\theta_1 = \frac{n_1 \omega}{c} \sin\theta_1\)

透過波: \(k_{2x} = k_2 \sin\theta_2 = \frac{n_2 \omega}{c} \sin\theta_2\)

3 位相整合

\(k_{1x} = k_{2x}\) より：

\[ n_1 \sin\theta_1 = n_2 \sin\theta_2 \]

4.2.3 フレネルの公式

界面での反射率と透過率は、入射波の偏光状態（s偏光：電場が入射面に垂直、p偏光：電場が入射面内）によって異なる。

フレネルの公式

s偏光（TE波）：

\[ r_s = \frac{n_1\cos\theta_1 - n_2\cos\theta_2}{n_1\cos\theta_1 + n_2\cos\theta_2} \] \[ t_s = \frac{2n_1\cos\theta_1}{n_1\cos\theta_1 + n_2\cos\theta_2} \]

p偏光（TM波）：

\[ r_p = \frac{n_2\cos\theta_1 - n_1\cos\theta_2}{n_2\cos\theta_1 + n_1\cos\theta_2} \] \[ t_p = \frac{2n_1\cos\theta_1}{n_2\cos\theta_1 + n_1\cos\theta_2} \]

垂直入射（\(\theta_1 = 0\)）の場合、両偏光で同じ反射率：

\[ R = \left|\frac{n_1 - n_2}{n_1 + n_2}\right|^2 \]

例題4.2: ガラスの反射率

問題: 屈折率 n = 1.5 のガラスに空気中から垂直に光を入射させたとき、1枚のガラス板（両面）での反射による光損失は何%か。

解答:

空気-ガラス界面での反射率：

\[ R_1 = \left(\frac{1 - 1.5}{1 + 1.5}\right)^2 = \left(\frac{-0.5}{2.5}\right)^2 = 0.04 = 4\% \]

ガラス-空気界面でも同様に 4%が反射される。

第1面を透過した光は \(T_1 = 1 - R_1 = 0.96\)

第2面を透過した光は \(T_2 = 0.96 \times (1 - 0.04) = 0.96^2 = 0.922\)

全損失: \(1 - 0.922 = 0.078 \approx 7.8\%\)

多重反射を無視した近似では約8%の光が反射で失われる。これがレンズに反射防止コートを施す理由である。

4.2.4 ブリュースター角

p偏光の反射率がゼロになる入射角をブリュースター角という。

ブリュースター角
\[ \tan\theta_B = \frac{n_2}{n_1} \]

このとき、反射波と屈折波が直交する：\(\theta_1 + \theta_2 = 90°\)

ブリュースター角の物理的解釈：屈折波の電場（p偏光の場合、入射面内）の振動方向が、反射波の進行方向と一致する。振動する双極子はその軸方向には放射しないため、反射波が発生しない。

コラム: 偏光サングラスとブリュースター角

水面や道路からの反射光（ぎらつき）は、ブリュースター角付近で入射した光が多いため、主にs偏光成分（水平偏光）である。偏光サングラスはs偏光を遮断するフィルターを内蔵しており、反射光のまぶしさを効果的に低減する。

空気-水界面のブリュースター角：

\[ \theta_B = \arctan(1.33) = 53° \]

この角度は太陽高度約37°に対応し、日中の多くの時間帯で水面反射がほぼs偏光となる。

写真撮影でも偏光フィルターを使用して、ガラス越しの反射や水面の反射を除去することができる。

4.2.5 全反射

屈折率の大きい媒質から小さい媒質へ光が進むとき、入射角がある臨界角 \(\theta_c\) を超えると、光は完全に反射される。

全反射の条件
\[ \sin\theta_c = \frac{n_2}{n_1} \quad (n_1 > n_2) \]

臨界角以上の入射角で、反射率は100%（透過率0%）となる。

全反射時でも電磁場は第2媒質側にわずかに浸入する。これをエバネッセント波という：

\[ E_2 \propto e^{-z/\delta} \quad \text{ここで} \quad \delta = \frac{\lambda}{2\pi\sqrt{n_1^2\sin^2\theta_1 - n_2^2}} \]

エバネッセント波の浸入深さ \(\delta\) は波長程度である。

例題4.3: 光ファイバーの開口数

問題: コア（屈折率 \(n_1 = 1.48\)）とクラッド（屈折率 \(n_2 = 1.46\)）からなる光ファイバーについて、(a) 臨界角、(b) 開口数 NA を求めよ。

解答:

(a) 臨界角：

\[ \sin\theta_c = \frac{n_2}{n_1} = \frac{1.46}{1.48} = 0.9865 \] \[ \theta_c = \arcsin(0.9865) = 80.6° \]

(b) 開口数（ファイバーに入射できる最大角度の正弦）：

\[ NA = \sqrt{n_1^2 - n_2^2} = \sqrt{1.48^2 - 1.46^2} = \sqrt{2.1904 - 2.1316} = \sqrt{0.0588} = 0.24 \]

これは入射角 \(\theta_{max} = \arcsin(0.24) = 14°\) に対応する。この角度より大きな入射角の光はファイバー内で全反射条件を満たさず、漏れ出してしまう。

4.3 吸収のメカニズム

4.3.1 ローレンツ振動子モデル

物質による光吸収の古典的モデルとして、電子を調和振動子として扱うローレンツモデルがある。電子は原子核に束縛され、固有振動数 \(\omega_0\) を持つと仮定する。

調和振動子の運動方程式
\[ m\ddot{x} + m\gamma\dot{x} + m\omega_0^2 x = -eE_0 e^{-i\omega t} \]

\(\gamma\): 減衰定数（摩擦による損失）

\(\omega_0\): 固有振動数（復元力を特徴づける）

定常解 \(x = x_0 e^{-i\omega t}\) を仮定すると：

\[ x_0 = \frac{-eE_0/m}{\omega_0^2 - \omega^2 - i\gamma\omega} \]

導出: ローレンツモデルの誘電関数

1 分極の計算

N個の振動子による分極（単位体積あたり）：

\[ P = -Nex = \frac{Ne^2/m}{\omega_0^2 - \omega^2 - i\gamma\omega} E \]
2 電気感受率

\(P = \epsilon_0 \chi E\) より：

\[ \chi(\omega) = \frac{Ne^2/(\epsilon_0 m)}{\omega_0^2 - \omega^2 - i\gamma\omega} = \frac{\omega_p^2}{\omega_0^2 - \omega^2 - i\gamma\omega} \]

ここで \(\omega_p = \sqrt{Ne^2/(\epsilon_0 m)}\) はプラズマ周波数。

3 複素誘電関数 \[ \tilde{\epsilon}(\omega) = \epsilon_0(1 + \chi) = \epsilon_0\left(1 + \frac{\omega_p^2}{\omega_0^2 - \omega^2 - i\gamma\omega}\right) \]
4 実部と虚部 \[ \epsilon'(\omega) = \epsilon_0\left(1 + \frac{\omega_p^2(\omega_0^2 - \omega^2)}{(\omega_0^2 - \omega^2)^2 + \gamma^2\omega^2}\right) \] \[ \epsilon''(\omega) = \epsilon_0 \frac{\omega_p^2 \gamma\omega}{(\omega_0^2 - \omega^2)^2 + \gamma^2\omega^2} \]

この結果から重要な特徴が読み取れる：

  • 共鳴周波数 \(\omega \approx \omega_0\) 付近で吸収（\(\epsilon''\)）が最大
  • 共鳴の幅は減衰定数 \(\gamma\) で決まる
  • \(\omega < \omega_0\) では \(\epsilon' > \epsilon_0\)（正常分散）
  • \(\omega > \omega_0\) では \(\epsilon' < \epsilon_0\) となりうる（異常分散）

4.3.2 様々な吸収過程

電磁波の周波数に応じて、異なる吸収メカニズムが支配的となる。

周波数領域 吸収メカニズム 典型的な物質 特性周波数
電波〜マイクロ波 配向分極（双極子回転） 水、極性分子 GHz帯
赤外線 分子振動（格子振動） CO₂、H₂O、結晶 10¹³〜10¹⁴ Hz
可視〜紫外 電子遷移 色素、半導体 10¹⁴〜10¹⁵ Hz
X線 内殻電子励起 重元素 10¹⁶〜10¹⁸ Hz

4.3.3 ランベルト-ベールの法則

吸収媒質を通過する光の強度減衰は、指数関数的に減少する。

ランベルト-ベールの法則
\[ I(z) = I_0 e^{-\alpha z} = I_0 \times 10^{-\varepsilon c z} \]

\(\alpha\): 吸収係数 [m⁻¹]

\(\varepsilon\): モル吸光係数 [L/(mol·cm)]

\(c\): 濃度 [mol/L]

吸光度: \(A = \log_{10}(I_0/I) = \varepsilon c z\)

例題4.4: 色素溶液の吸収

問題: ある色素のモル吸光係数が波長500 nmで \(\varepsilon = 5 \times 10^4\) L/(mol·cm) である。濃度 \(10^{-5}\) mol/L の溶液を1 cmのセルに入れたとき、透過率は何%か。

解答:

吸光度：

\[ A = \varepsilon c z = 5 \times 10^4 \times 10^{-5} \times 1 = 0.5 \]

透過率：

\[ T = 10^{-A} = 10^{-0.5} = 0.316 = 31.6\% \]

約70%の光が吸収される。この程度の吸光度が分光分析に適している。

コラム: 電子レンジと水分子

電子レンジ（2.45 GHz）は水分子の配向分極を利用して加熱を行う。水分子は永久双極子モーメントを持ち、電場方向に配向しようとする。交番電場の下では双極子が回転運動を繰り返し、周囲の分子との摩擦でエネルギーが熱に変換される。

なぜ2.45 GHzなのか？

  • 水の誘電緩和周波数（約20 GHz）より低い周波数を使用
  • 共鳴周波数そのものを使うと、表面でのみ吸収され内部まで加熱できない
  • 2.45 GHzでは浸透深さが数cmとなり、食品内部まで加熱可能
  • ISMバンド（産業・科学・医療用）の割り当て周波数

浸透深さ: \(\delta = c/(2\pi f \cdot 2\kappa) \approx 1.4\) cm（室温の水の場合）

4.4 金属の光学的性質

4.4.1 ドルーデモデル

金属中の自由電子の応答は、ドルーデモデルで記述される。これはローレンツモデルで復元力をゼロ（\(\omega_0 = 0\)）としたものに相当する。

ドルーデモデルの誘電関数
\[ \tilde{\epsilon}(\omega) = \epsilon_0\left(1 - \frac{\omega_p^2}{\omega^2 + i\gamma\omega}\right) \]

プラズマ周波数: \(\omega_p = \sqrt{\frac{Ne^2}{\epsilon_0 m^*}}\)（典型的な金属で紫外領域）

減衰定数: \(\gamma = 1/\tau\)（\(\tau\): 電子の緩和時間）

低周波極限（\(\omega \ll \gamma\)）では、DC導電率が回復される：

\[ \sigma_0 = \frac{\omega_p^2 \epsilon_0}{\gamma} = \frac{Ne^2\tau}{m^*} \]

4.4.2 金属の反射率

金属表面での高い反射率は、負の誘電率（\(\omega < \omega_p\) で \(\epsilon' < 0\)）に起因する。

導出: 金属の高反射率

1 損失を無視した場合

\(\gamma \rightarrow 0\) の極限で、\(\omega < \omega_p\) では：

\[ \epsilon = \epsilon_0\left(1 - \frac{\omega_p^2}{\omega^2}\right) < 0 \]
2 複素屈折率

\(n^2 = \epsilon/\epsilon_0 < 0\) より、屈折率は純虚数：

\[ n = i\kappa \quad \text{ここで} \quad \kappa = \sqrt{\frac{\omega_p^2}{\omega^2} - 1} \]
3 垂直入射の反射率 \[ R = \left|\frac{1 - n}{1 + n}\right|^2 = \left|\frac{1 - i\kappa}{1 + i\kappa}\right|^2 = \frac{1 + \kappa^2}{1 + \kappa^2} = 1 \]
4 損失がある場合

有限の \(\gamma\) では反射率は100%より若干低くなる。可視光での典型的な金属の反射率は90-99%。

4.4.3 スキンデプス

電磁波が金属内部に浸透できる深さ（スキンデプス）は周波数に依存する。

スキンデプス

低周波（\(\omega \ll \gamma\)）では：

\[ \delta = \sqrt{\frac{2}{\mu\sigma\omega}} \]

高周波（\(\omega \gg \gamma\)）では：

\[ \delta = \frac{c}{\omega_p} \]

例題4.5: 銅のスキンデプス

問題: 銅（導電率 \(\sigma = 5.96 \times 10^7\) S/m）のスキンデプスを、(a) 60 Hz、(b) 1 MHz、(c) 1 GHz で求めよ。

解答:

\(\delta = \sqrt{2/(\mu_0 \sigma \omega)} = \sqrt{2/(\mu_0 \sigma \cdot 2\pi f)}\)

(a) 60 Hz：

\[ \delta = \sqrt{\frac{2}{4\pi \times 10^{-7} \times 5.96 \times 10^7 \times 2\pi \times 60}} = 8.5 \text{ mm} \]

(b) 1 MHz：

\[ \delta = \sqrt{\frac{2}{4\pi \times 10^{-7} \times 5.96 \times 10^7 \times 2\pi \times 10^6}} = 66 \text{ μm} \]

(c) 1 GHz：

\[ \delta = \sqrt{\frac{2}{4\pi \times 10^{-7} \times 5.96 \times 10^7 \times 2\pi \times 10^9}} = 2.1 \text{ μm} \]

高周波になるほど電流は導体表面に集中する（表皮効果）。このため、高周波回路では導体表面の状態が重要になる。

4.4.4 プラズモン

金属中の自由電子の集団振動をプラズモンという。表面プラズモンは金属-誘電体界面に局在する電磁モードで、ナノフォトニクスで重要な役割を果たす。

コラム: 表面プラズモンとバイオセンサー

表面プラズモン共鳴（SPR）は、生体分子の検出に広く利用されている高感度センサー技術である。

原理：

  • 金薄膜表面で全反射条件付近の光を入射
  • 特定の角度（共鳴角）で表面プラズモンが励起され、反射率が低下
  • 表面近傍の屈折率変化（分子の吸着）で共鳴角がシフト
  • 10⁻⁶ RIU（屈折率単位）の変化を検出可能

応用：

  • 抗体-抗原反応のリアルタイムモニタリング
  • 薬物候補分子のスクリーニング
  • タンパク質間相互作用の解析

4.5 量子効果

4.5.1 光電効果

光電効果は、光が金属表面に当たると電子が放出される現象である。この現象は古典電磁気学では説明できず、光の粒子性（光量子仮説）の証拠となった。

光電効果の方程式（アインシュタイン、1905年）
\[ K_{max} = h\nu - W = h\nu - h\nu_0 \]

\(K_{max}\): 放出電子の最大運動エネルギー

\(W = h\nu_0\): 仕事関数（電子を放出するのに必要な最小エネルギー）

\(\nu_0\): 閾周波数（これ以下の周波数では電子放出なし）

光電効果の特徴と古典論との矛盾：

観測事実 古典論の予測 量子論の説明
閾周波数の存在 どんな周波数でも十分な強度なら電子放出 \(h\nu < W\) では光子1個のエネルギーが不足
即時的な電子放出 弱い光では電子がエネルギーを蓄積するのに時間がかかる 光子1個が瞬時にエネルギーを与える
電子エネルギーは光強度に依存しない 強い光ほど高エネルギーの電子 光子のエネルギーは周波数のみで決まる
電子数は光強度に比例 説明可能 光子数が強度に比例

例題4.6: 光電効果の実験

問題: セシウム（仕事関数 W = 2.1 eV）に波長400 nmの光を照射した。(a) 放出電子の最大運動エネルギー、(b) 阻止電圧を求めよ。

解答:

(a) 光子エネルギー：

\[ E = \frac{hc}{\lambda} = \frac{1240 \text{ eV·nm}}{400 \text{ nm}} = 3.1 \text{ eV} \]

最大運動エネルギー：

\[ K_{max} = E - W = 3.1 - 2.1 = 1.0 \text{ eV} \]

(b) 阻止電圧（電子を止めるのに必要な電圧）：

\[ eV_s = K_{max} \] \[ V_s = \frac{K_{max}}{e} = \frac{1.0 \text{ eV}}{e} = 1.0 \text{ V} \]

4.5.2 コンプトン散乱

X線が電子に散乱されるとき、波長が長くなる（エネルギーが減少する）現象。光子の粒子性の直接的な証拠である。

コンプトン散乱の波長変化
\[ \Delta\lambda = \lambda' - \lambda = \frac{h}{m_e c}(1 - \cos\theta) = \lambda_C(1 - \cos\theta) \]

\(\lambda_C = h/(m_e c) = 2.43 \times 10^{-12}\) m：コンプトン波長

\(\theta\): 散乱角

導出: コンプトン散乱の公式

1 光子と電子の衝突を考える

入射光子（エネルギー \(E = h\nu\)、運動量 \(p = h/\lambda\)）が静止電子に衝突。

2 エネルギー保存 \[ h\nu + m_e c^2 = h\nu' + \sqrt{(m_e c^2)^2 + (p_e c)^2} \]
3 運動量保存（x成分、y成分） \[ \frac{h}{\lambda} = \frac{h}{\lambda'}\cos\theta + p_e\cos\phi \] \[ 0 = \frac{h}{\lambda'}\sin\theta - p_e\sin\phi \]
4 \(\phi\) を消去して整理

3つの方程式から電子の運動量 \(p_e\) と反跳角 \(\phi\) を消去すると：

\[ \lambda' - \lambda = \frac{h}{m_e c}(1 - \cos\theta) \]

4.5.3 光子吸収と遷移確率

量子力学によると、光子の吸収確率はフェルミの黄金律で与えられる：

遷移確率（フェルミの黄金律）
\[ W_{fi} = \frac{2\pi}{\hbar} |\langle f | \hat{H}' | i \rangle|^2 \rho(E_f) \]

\(\hat{H}' = -e\vec{r} \cdot \vec{E}\): 摂動ハミルトニアン（双極子近似）

\(\rho(E_f)\): 終状態の状態密度

選択則：行列要素 \(\langle f | \vec{r} | i \rangle\) が非ゼロとなる条件から、許容遷移と禁制遷移が決まる。

コラム: 光電効果とノーベル賞

アインシュタインは1905年の論文「光の発生と変換に関する発見的な観点について」で光量子仮説を提唱した。これは同年の特殊相対性理論やブラウン運動の論文とともに「奇跡の年」の業績として知られる。

興味深いことに、アインシュタインが1921年のノーベル物理学賞を受賞したのは相対性理論ではなく、光電効果の法則の発見に対してであった。当時、相対性理論は革命的すぎて、まだ十分に検証・受容されていないと考えられていたためである。

光電効果に関連するノーベル賞：

  • 1905年: レーナルト（光電効果に関する研究）
  • 1921年: アインシュタイン（光電効果の法則の発見）
  • 1923年: ミリカン（電気素量と光電効果の研究）

4.6 散乱現象の詳細

4.6.1 散乱の分類

電磁波の散乱は、粒子サイズと波長の関係、およびエネルギーの変化の有無によって分類される。

散乱の種類 条件 特徴 例
レイリー散乱 \(d \ll \lambda\) \(I \propto 1/\lambda^4\)、弾性散乱 青空、チンダル現象
ミー散乱 \(d \sim \lambda\) 波長依存性弱い、前方散乱優勢 雲、霧、コロイド
幾何光学散乱 \(d \gg \lambda\) 反射・屈折で記述可能 虹、レンズ効果
ラマン散乱 非弾性散乱 周波数シフト、分子振動の情報 分子分光
ブリルアン散乱 非弾性散乱 音響フォノンとの相互作用 弾性率測定

4.6.2 ラマン散乱

ラマン散乱は、光子が分子と相互作用する際に分子の振動・回転状態が変化し、散乱光の周波数がシフトする現象である。

ラマン散乱

ストークス散乱: \(\nu_s = \nu_0 - \nu_v\)（エネルギーを分子に与える）

アンチストークス散乱: \(\nu_{as} = \nu_0 + \nu_v\)（エネルギーを分子から受け取る）

\(\nu_0\): 入射光周波数、\(\nu_v\): 分子振動周波数

ラマンシフト \(\Delta\nu = \nu_v\) は分子に固有の値であり、分子の「指紋」として同定に利用される。

例題4.7: ラマン分光による分子同定

問題: 波長532 nmのレーザーでCO₂分子のラマンスペクトルを測定したところ、ストークス線が波長553 nmに観測された。CO₂の対称伸縮振動の周波数を求めよ。

解答:

入射光の波数：

\[ \bar{\nu}_0 = \frac{1}{\lambda_0} = \frac{1}{532 \times 10^{-7} \text{ cm}} = 18797 \text{ cm}^{-1} \]

ストークス線の波数：

\[ \bar{\nu}_s = \frac{1}{553 \times 10^{-7}} = 18083 \text{ cm}^{-1} \]

ラマンシフト：

\[ \Delta\bar{\nu} = \bar{\nu}_0 - \bar{\nu}_s = 18797 - 18083 = 714 \text{ cm}^{-1} \]

振動周波数：

\[ \nu_v = c \times \Delta\bar{\nu} = 3 \times 10^{10} \times 714 = 2.14 \times 10^{13} \text{ Hz} \]

これはCO₂の対称伸縮振動（フェルミ共鳴のため2つのピークに分裂するうちの一方）に対応する。

4.6.3 散乱断面積

散乱の強さは散乱断面積で定量化される。

散乱断面積
\[ \sigma = \frac{\text{散乱パワー}}{\text{入射強度}} \quad [\text{m}^2] \]

レイリー散乱断面積（小粒子）：

\[ \sigma_R = \frac{8\pi}{3}\left(\frac{2\pi}{\lambda}\right)^4 \left|\frac{n^2-1}{n^2+2}\right|^2 a^6 \]

\(a\): 粒子半径、\(n\): 粒子の相対屈折率

コラム: ラマン分光の応用

ラマン分光は非破壊・非接触で分子構造の情報が得られる強力な分析手法である。

主な応用分野：

  • 製薬: 原薬・製剤の同定、結晶多形の判別
  • 半導体: 応力測定、結晶性評価
  • 考古学: 顔料・材料の非破壊分析
  • 医療: がん組織の識別（術中診断）
  • セキュリティ: 爆発物・薬物の検出
  • 宇宙探査: 火星探査車での鉱物分析

近年は表面増強ラマン散乱（SERS）により、単分子レベルの検出も可能になっている。

問題4.3 発展

ローレンツ振動子モデルにおいて、共鳴周波数 \(\omega_0 = 5 \times 10^{15}\) rad/s、プラズマ周波数 \(\omega_p = 10^{16}\) rad/s、減衰定数 \(\gamma = 5 \times 10^{13}\) rad/s の物質について：(a) \(\omega = 0.9\omega_0\) での複素屈折率、(b) \(\omega = \omega_0\) での吸収係数、(c) 異常分散が起こる周波数範囲を求めよ。

問題4.4 考察

光電効果では、光の強度を上げても放出電子の最大エネルギーは変化しない。一方、波長を短くすると最大エネルギーは増加する。これらの事実を、(a) 古典電磁気学、(b) 光量子仮説のそれぞれの立場から説明を試み、古典論が失敗する理由を論じよ。

問題4.5 総合

真空中から屈折率 n = 1.33 の水中に、s偏光とp偏光の光を入射角60°で入射させた。(a) 各偏光の反射率と透過率を計算せよ。(b) 反射光の偏光状態（s偏光とp偏光の振幅比）を求めよ。(c) この結果から、水面からの反射光を偏光サングラスで低減できる理由を説明せよ。

第4回のまとめ

学習した主要概念

  • 物質中の電磁波は複素屈折率で記述され、実部が伝播速度、虚部が吸収を決める
  • フレネルの公式により界面での反射・透過が計算でき、偏光依存性がある
  • ローレンツ/ドルーデモデルで誘電体/金属の光学的性質を理解できる
  • 光電効果、コンプトン散乱は光の粒子性を示す量子効果である
  • 散乱は粒子サイズと波長の関係で分類され、レイリー、ミー、ラマンなどがある

重要公式

スネルの法則: \(n_1 \sin\theta_1 = n_2 \sin\theta_2\)

垂直入射反射率: \(R = \left|\frac{n_1 - n_2}{n_1 + n_2}\right|^2\)

ブリュースター角: \(\tan\theta_B = n_2/n_1\)

光電効果: \(K_{max} = h\nu - W\)

コンプトン散乱: \(\Delta\lambda = \lambda_C(1 - \cos\theta)\)
